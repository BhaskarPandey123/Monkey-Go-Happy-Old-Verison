# Monkey Go Happy Inspired Game Prototype.

This project is a true inspiration and serves as a practical test of my skills in condition nesting and other abilities
I've developed through self-study. I am still exploring the world of digital development and hope that this prototype will
Provide me with a solid foundation to collaborate with experienced developers.
## Features
- Banana generation
- Stone generation
- Grass generation
- Banana collision
- Score increment
- Score Multiplier for eating bananas
- Game States:
  - `PLAY`
  - `END`
- Start and restart buttons to change game states.

## Installation 

1. Clone the repository:
```bash
   git clone https://github.com/BhaskarPandey123/Monkey-Go-Happy-Old-Verison
```
2. Navigate into the project folder:
```bash
   cd Monkey-Go-Happy-Old-Verison
```

3. Open `index.html` in your browser.
